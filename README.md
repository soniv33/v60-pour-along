# V60 Pour Along

A guided brew timer for James Hoffmann's V60 "better" technique. Step through each pour, wait, and swirl with a live countdown, animated ring, and audio cues — no guesswork, no distractions.

**[Open the app →](https://soniv33.github.io/v60-pour-along)**

## Features

- **7-step guided recipe** — bloom, swirl, first pour, final pour, swirl, drawdown
- **Three dose sizes** — 15 g / 250 ml, 22 g / 370 ml, 30 g / 500 ml
- **Per-step countdown** with an animated ring; drawdown counts up so you're not rushed
- **Tap to pause / resume** on the timer circle
- **Audio + haptic cues** on each step transition
- **Brew history** — last brew time and verdict saved locally
- **Installable PWA** — add to home screen for a full-screen, offline experience
- **Light and dark mode** — follows your system preference

## Recipe (1-cup / 15 g)

| Step | Action | Target |
|------|--------|--------|
| 1 | Bloom pour | → 50 g (12 s) |
| 2 | Swirl | Saturate grounds (8 s) |
| 3 | Bloom wait | De-gas (25 s) |
| 4 | First pour | → 150 g (30 s) |
| 5 | Final pour | → 250 g (30 s) |
| 6 | Swirl | Level the bed (8 s) |
| 7 | Drawdown | Drain fully, tap Finish when dry |

Target total brew time: **3:00 – 3:30**

## Install as PWA
